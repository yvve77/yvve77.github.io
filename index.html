I'm a freshman at UIUC double-majoring in CS and Bioengineering. I picked this combination on purpose — one major asks how to build systems, the other asks how living ones work. I like having both questions open at the same time.

Before college, I published a paper on biosensors for tumor monitoring, spent time at a genomics lab tracing variants through family DNA, and led a campus biodiversity project that ended up involving GIS maps and conservation algorithms. Most of these started as curiosity and became something bigger.

I built DDL Collector because I kept almost missing deadlines. It now sends me a morning email every day with exactly what's left to do. Small problem, concrete fix — that's usually how I end up with side projects.

Featured Project
DDL Collector
Python · GitHub Actions · HTML / JS · GitHub Pages
Automatically tracks assignment deadlines across UIUC courses — sends a daily 8 AM email with only incomplete tasks, and hosts a live dashboard where you can check things off. No manual tracking, no missed deadlines.
  • Daily email digest to @illinois.edu, filtered to incomplete assignments only
  • Monday banner reminding BioE 210 is due that day
  • Interactive dashboard — check off tasks, state persists in tasks.json
  • Color-coded by course; auto-committed via GitHub Actions on each run
